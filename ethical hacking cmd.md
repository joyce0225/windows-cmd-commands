# 🛠️ Essential Windows CMD Commands for Ethical Hackers

## 📌 Basic CMD Operations
1. **Open CMD as Administrator**: Press Windows + S and type CMD, then right-click and choose 'Open as administrator'. 🖱️
2. **`help`**: Lists all available CMD commands. 📜
3. **`command /?`**: Provides help for specific commands. ❓

## 🎨 Customizing CMD
4. **`title [text]`**: Changes the title of the CMD window. ✏️
5. **`color [attributes]`**: Changes the background and text color of CMD. 🎨

## 💻 System Information and Management
6. **`systeminfo`**: Displays detailed system information. 💾
7. **`prompt [text]$G`**: Changes the command prompt text. ✏️
8. **`del %temp%\*`**: Deletes all temporary files to free up space. 🧹
9. **`wmic product get name`**: Lists all installed programs. 📦
10. **`driverquery`**: Displays all installed drivers. 🚗

## 📂 File and Directory Management
11. **`tree`**: Shows directory paths and files in a tree structure. 🌲
12. **`attrib +h +s +r [folder]`**: Hides a folder. 🙈
13. **`attrib -h -s -r [folder]`**: Unhides a folder. 👀

## 🌐 Network and Connectivity
14. **`ipconfig /all`**: Displays detailed network configuration. 🌍
15. **`nslookup`**: Looks up DNS server addresses and maps IP addresses. 🕵️‍♂️
16. **`netstat -a`**: Shows all active connections and listening ports. 🔌
17. **`ping [address]`**: Tests network connectivity to a specific address. 🏓
18. **`tracert [address]`**: Traces the route packets take to a destination. 🛣️

## 📶 Wi-Fi Management
19. **`netsh wlan show profile`**: Lists Wi-Fi networks the PC has connected to. 📡
20. **`netsh wlan show profile [network] key=clear`**: Reveals the Wi-Fi password in clear text. 🔑

## 🧹 Cleaning and Resetting
21. **`cls`**: Clears the Command Prompt screen. 🧼
22. **`systemreset -factoryreset`**: Resets the PC to factory setup. 🔄

## 🆕 Additional Features
23. **`qrencode -d / [address]`**: Creates a QR code for a specific website. 📱

To close the CMD window, type `exit`.

These commands are essential for ethical hackers to manage, troubleshoot, and gather information using Windows Command Prompt. 💻🔐
